# Uptime Globe MVP (Front Page Only)

This repository now contains a **front-end only MVP** for Uptime Globe.

## What's included

- A single polished landing page built with Next.js App Router.
- TailwindCSS styling.
- No backend services, database, authentication, cron jobs, or API routes.

## Why this version?

This structure is intentionally minimal so deployment to Vercel is straightforward with no extra environment setup.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deploy to Vercel

1. Push this repo to GitHub.
2. Import it in Vercel.
3. Deploy with default settings.

No required environment variables for this MVP.
